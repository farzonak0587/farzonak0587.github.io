Movie Webpage Project

Context

I had to collaborate with a partner to develop an imaginary movie website using our understanding of basic HTML and CSS for this SEP Movie Project. After the release of Inside Out 2, my partner and I thought it would be logical to create a movie about Inside Out 3.

Content

We had to apply the HTML and CSS knowledge we have gained in order to develop a webpage. Some requoremtns for HTML and CSS were:

HTML

  • Headings
  • paragraphs
  • divs
  • spans
  • inputs
  • submit button
  • customized Google fonts

CSS

  • Background color
  • margins
  • paddings
  • borders
  • CSS selectors such as element / class / id
  • two local images

Additionally, he offered us the opportunity to earn extra credit by assigning us the task to make a sequel to our initial story using the same CSS. My partner and I introduced a new feeling called "Longing" into the movie, which basically changes Riley's (the main character) life.

Reflection

Challenges

My partner and I had several difficulties, including a lack of communication skills and our ideas were also different. We were unsure of how to proceed. We didn't communicate at all on the first day when we were meant to plan our project. We spoke on the second day, but it was a bit awkward and we weren't entirely sure what we were attempting to accomplish for the project. We began gathering all the necessary materials for the project after determining what needed to be done. After that, we basically did our bit at home and didn't really talk to each other.I also have trouble using most of my class time efficiently because I get easily distracted and constantly believe I could complete the task at home. Even if I do it at home, I feel compelled to do it rather than enjoying it because I don't want to do anything at home.

Takeaways

  • try to communicate !
  • ask teachers or people around you for help
  • Plan out what you want to do first
  • make sure you meet all the requirements
  • Try to use your class time efficiently

Next Steps

I want to strive to make my work more appealing and more complex in the future.
